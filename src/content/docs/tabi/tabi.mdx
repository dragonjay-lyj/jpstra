## 足袋的穿着感

足袋拥有一种独特的穿着感。当穿上一双完美贴合脚型、面料亲肤的足袋时，那种惬意感，足以媲美系紧角带[^1]时的那份舒爽，能让整个人的精神为之一振。这种感觉，是穿普通袜子绝对无法体验的。甚至有人会说，自己正是为了穿足袋，才特意换上和服。此外，足袋也扮演着让和服身姿更显挺拔干练的重要角色，是不可或缺的存在。可以说，足袋并非内衣的一部分，在和装的语境下，它毋宁说是着物本身的一部分。

[^1]: 角带 (Kakuobi): 日本男性和服（如着物、浴衣）所使用的一种较硬、较窄的腰带，能将衣物固定并塑造出挺拔的腰部线条。

说到男士足袋，一般认为礼服用的是白足袋，其他场合则用黑足袋或绀色[^2]足袋。但您是否知道，实际上足袋的种类远比这要丰富得多呢？它绝非“足袋嘛，哪种不都一样”这么简单。仅仅是换一双足袋，和装的整体观感就会为之骤变，更重要的是，心境也会焕然一新。在此，除了演艺界至今仍在使用的特殊足袋外，我们将主要介绍普通大众也能入手的足袋信息。

[^2]: 绀色 (Kon-iro): 指一种非常深的、近乎黑色的海军蓝。是日本传统色彩之一，在男性服饰中尤为常见，代表沉稳与格调。

## 何为一双称心如意的足袋？

所谓“好穿的足袋”，其实因人而异，以下所述终究不过是我的个人偏好。但总的来说，我认为理想的足袋应具备以下特质：“**面料为加厚白棉布[^6]，内衬用漂白棉[^7]，配四枚扣襻[^8]，尺寸略微偏紧，而凌驾于这一切之上的，是必须完美贴合自己的脚型。**”

[^6]: 加厚白棉布 (キャラコ木綿 / Calico Cotton): 指一种质地厚实、坚韧耐用的平纹白棉布。因其挺括有型且耐磨，是制作高品质足袋的经典面料。
[^7]: 漂白棉 (晒 / Sarashi): 指经过漂白处理的柔软棉布，常用作足袋的内衬（晒裏 / sarashi-ura）。其优点是亲肤、吸汗，能提供舒适的穿着感。
[^8]: 扣襻 (小鉤 / Kohaze): 指足袋脚踝内侧用于固定的金属或骨制挂扣。其数量（通常为三至五枚）不仅影响贴合度，也与礼仪规制有关，枚数多者更为正式。

诚然，最理想的莫过于在专业的足袋店量脚定制[^9]一双。但在入门之初，我更推荐您多试穿比较各家店铺的现成品，从中找到最合自己心意的那一款。不同店铺的足袋，即便尺码相同，尺寸也常有微妙差异，且各有其韵味，穿着感亦不尽相同，因此光是试穿比较的过程，本身就是一种乐趣。

[^9]: 量脚定制 (誂え / Atsurae): 日本传统手工艺中的“定制”或“量身定做”。在足袋的语境下，指足袋师傅会测量客人的脚型数据，制作专属的版型，以达到最完美的贴合度。这被视为穿着足袋的最高享受。

定制足袋，快则一月，慢则可能耗时半年以上。这是因为要得到一双真正合脚的定制足袋，中途还需要到店里试穿确认，勤跑几趟足袋店是免不了的。而那些被称为“老字号”的足袋店，不知为何，大多集中在东京或京都一带，我所居住的地方小城几乎没有，因此很难有机会去定制。

仅供参考，在足袋店里，现成品的售价大约在一双3,000至4,000日元上下。定制的价格，通常比现成品高出一到两成。关于定制，各家店铺规定不同，但通常接受十双起的订单。价格虽然略高，但任何一家老店定制出的足袋，其穿着感都有云泥之别，请务必尝试一次。一双3000日元的足袋确实不便宜，但若保养得当，也能穿上相当长的时间；再考虑到那无与伦比的穿着感，我便自顾自地认定，这价格尚属公道。

**白足袋（左：京都“分铜屋”[^10]出品，右：浅草“めうがや”出品）**

[^10]: 分铜屋 (Fundōya): 位于京都的著名和装老铺，与“めうがや”齐名，以其高品质的足袋和其他和装小物而受到行家推崇。

在弓道等领域，白足袋据说是作为制服的一部分被强制要求穿着的。但对于日常穿着和服的普通场合而言，通常并不会穿白足袋。当然，这并非意味着“不可以穿”。尤其是在夏季，白足袋反而能让整体造型显得清爽利落，更具凉意。

白足袋是搭配第一礼装（黑纹付羽织袴[^11]）等服饰，在极为正式的场合下穿着的、等级最高的足袋。反过来说，将白足袋用于日常穿着，也被视为一种极致的奢华。这大概是因为白足袋极易弄脏，真正讲究仪容的人，一天之内甚至需要更换数次。更何况，穿着脏污的足袋出现在人前也是一种失礼，因此，在日常生活中选择无需太过费心打理的彩色足袋，似乎已成为普遍的做法。

[^11]: 黑纹付羽织袴 (黒紋付羽織袴 / Kuromontsuki Haori Hakama): 日本男性的最高级别和服礼装，由染有家纹的黑色和服（纹付）、外褂（羽织）和裙裤（袴）组成。是婚礼、葬礼等最庄重场合的指定着装。白足袋是其标准搭配。

曾任日本首相、被誉为“和服达人”的吉田茂[^12]，便以只穿白足袋而闻名，甚至被人起了个“白足袋”的绰号。或许在某些情境下，白足袋确实能营造出一种独特的威严与风骨。的确，仅仅是在脚下配上一双纯白的足袋，本身就极为引人注目，能瞬间成为视线的焦点。因此，我的看法是，在现代，除非您是出于个人执念而特意选择白足袋，否则一般只需在穿着带有“一之纹”[^13]以上等级的纹付等正式礼服时搭配白足袋即可。其他场合，即便是穿着袴装，根据个人喜好选择彩色足袋也未尝不可。

[^12]: 吉田茂 (Yoshida Shigeru, 1878-1967): 日本战后时期极具影响力的政治家，曾五次出任日本首相。他以其强硬的政治手腕和对传统和服的钟爱而闻名，其标志性的“白足袋”形象，成为他个人风格乃至一个时代印记的象征。
[^13]:一之纹 (一つ紋 / Hitotsu-mon): 指和服上染有一个家纹的款式。根据家纹的数量（一、三、五），和服的正式程度也不同，五个纹（五つ紋 / Itsutsu-mon）为最高级别。带有家纹的和服（纹付）通常用于较为正式的场合。

## 关于日常穿着的足袋

那么，说到日常穿着的足袋，自古便有“关东尚绀棉，关西崇黑缎”¹的说法。这其中所说的“黑繻子”（Kurojusu / Kuroshusu），即那种泛着黑色光泽的缎面足袋，恐怕是最大众化的男士足袋了，但我个人却无论如何也喜欢不起来。在我看来，脚下泛着油光锃亮的黑色，与其说格调高雅，倒不如说给人一种土气²之感，尤其与袴装搭配时，更显不伦不类。

若论颜色，我个人认为，绀色足袋仍是最佳选择。除了最正式的场合，它几乎是万能百搭的，能与绝大多数的和服相配。虽说是绀色，但如左图所示，许多款式在视觉上已十分接近黑色，但其色调实则千变万化，大可根据自己的喜好来挑选。绀色足袋会随着每一次洗涤，色泽越发有韵味³，穿着感也愈加舒适。不过，外出见人时，还是尽可能穿簇新一些的为好。

## 关于彩色足袋

男士足袋中，同样有彩色或带图案的款式。不过在男士足袋的语境下，由于黑、绀两色已是标准色，因此“彩色足袋”一词，通常指代这两种颜色以外的足袋。当然，严格来说，黑足袋与绀足袋也属于彩色足袋的范畴。女性足袋自古以来便色彩斑斓，而男士足袋的种类，近年来也变得相当丰富。

需要明确的是，无论男女，彩色足袋都纯粹是为了时尚穿搭。如今的彩色足袋，种类远比过去丰富，可以享受到根据和服或袴的颜色进行搭配的乐趣。似乎也有和服店会推荐顾客尝试用彩色足袋打造时髦感，但实际上，若非在色彩搭配上投入十二分的小心，一不小心就会显得非常不伦不类，务必当心。“只看着足袋本身觉得不错，结果一搭配和服与袴，才发现‘总觉得哪里不对劲’”——这样的翻车案例屡见不鲜。

那么，究竟有哪些彩色足袋可供选择呢？市面上常见的有比绀色更明亮的靛蓝色系、莺色等绿色系、栗色、深棕、浅棕等棕色系，以及灰色系等。此外，不同店铺的色调也大相径庭。例如，同为绀色，京都“分铜屋”的出品是沉稳的铁绀色，而东京新富町“大野屋”则有更明亮的蓝绀色可供选择，至于同在东京的浅草“めうがや”，更有色调再浅一阶的绀色，不一而足。

此外，足袋底的颜色也并非清一色的白色。有为绀色或深棕色足袋搭配纯黑鞋底的，也有为绀色足袋搭配浅蓝灰色鞋底、为鼠灰色足袋搭配浅灰色鞋底等，即鞋底采用与鞋面同色系的浅色设计。顺带一提，像舞台上的“黑子”[^14]常穿的那种黑棉布配黑底的足袋，俗称“乌足袋”[^15]。

[^14]:黑子 (黒子 / Kuroko): 日本歌舞伎等传统戏剧中，身着黑衣黑头套、负责在台上布置场景或辅助演员的舞台工作人员。他们被约定俗成地视为“不存在”，其服装即为“乌足袋”的典型应用场景。
[^15]:乌足袋 (烏足袋 / Karasu-tabi): 指鞋面与鞋底均为黑色的足袋，因其通体漆黑如乌鸦而得名。

另一方面，带图案的足袋，绝大多数是用于舞蹈等舞台表演。但也不乏条纹棉布足袋，或是在深艾草绿底色上印有松针图案、在靛蓝底色上印有麻叶纹[^16]图案的“小纹”[^17]足袋等，种类其实相当丰富。似乎还有足袋店，能用客人自己带来的和服边角料[^18]，为其定制心仪的足袋。不过，据说用和服面料做的足袋穿不了太久。

[^16]:麻叶纹 (麻の葉柄 / Asanoha-gara): 日本传统几何纹样之一，由六边形组合而成，形似大麻叶片。因大麻生长迅速且强韧，此纹样被赋予了“健康成长、祛病消灾”的美好寓意，是经典的吉祥图案。
[^17]:小纹 (小紋 / Komon): 指布满细小重复花纹的图案，以及使用这种面料制作的和服。在和服等级中属于日常便服，风格多样。
[^18]:边角料 (端切れ / Hagire): 指制作和服等衣物后剩余的布料。在日本，出于“勿体無い”（Mottainai，珍惜事物、反对浪费）的传统观念，这些边角料常被用来制作发饰、钱包、足袋等小物件。

## 关于足袋的材质等

足袋的材质有棉、麻、别珍[^19]、丝绸、化纤等，但最普及的，当属加厚白棉布（キャラコ）。价格低廉的涤纶等材质的足袋虽然打理方便，但质地过于柔软，容易显露出脚趾的形状。此外，如今市面上仍有皮质足袋，但并不常见，我也从未穿过。据说皮足袋穿上一天，气味会相当惊人。

[^19]:别珍 (別珍 / Velveteen): 一种仿天鹅绒的棉织物，表面有短而密的绒毛，手感柔软，富有光泽。在和装中，常用于制作冬季的羽织外褂或足袋，给人以温暖华丽之感。

关于足袋的内衬，通常是夏季用漂白棉（晒裏），冬季用绒布（ネル裏）等。但我个人即便是隆冬时节，也只穿漂白棉内衬的足袋。绒布内衬的足袋虽然触感不错，但我总嫌它过于厚重臃肿，不怎么穿。不过，也有像能剧演员[^20]那样，一年四季只穿绒布内衬足袋的人。这大概是因为绒布内衬能更好地吸收脚部湿气，并且能让脚部线条显得更为柔和。

[^20]:能剧演员 (能役者 / Nōyakusha): 日本传统戏剧“能剧”的表演者。他们在舞台上需要进行大量被称为“摺り足”（Suriashi）的、脚不离地的滑步动作，因此对足袋的贴合度、吸湿性和线条感有极高要求。

扣襻（小鉤）的数量，我认为还是四枚为最佳。三枚的款式，总觉得容易露出脚踝的皮肤，令人在意；五枚的又会妨碍坐姿，在穿着便服（着流し）[^21]时，更显得有些土气。据说现代甚至有六枚扣襻的款式，但归根结底，这也可根据个人喜好来选择。下图便是一双五枚扣襻的彩色足袋，是浅草“めうがや”的出品。

[^21]:着流し (着流し / Kinagashi): 指男性在日常非正式场合，只穿长着的物（長着），系上腰带，而不穿外褂（羽织）或裙裤（袴）的简便穿法。是一种轻松、随性的和装风格。

**正在扣上扣襻**

尺寸方面，建议选择与自己鞋码相同，或小0.5厘米的尺码。略微紧一些的足袋，褶皱不那么明显，习惯之后反而更好穿。但是，切忌勉强自己长时间穿着过紧的足D袋，这样不仅会很快磨破足袋，还有可能伤到脚。（本人有惨痛经验^^;）足袋的尺寸，现在普遍使用厘米（cm）单位，但在老字号的足袋店里，至今仍有很多店铺在使用“文”[^22]作为单位。顺带一提，之所以用“文”作单位，据说是源于古时用并排的“一文钱”的数量来测量长度的习俗。

[^22]:文 (文 / Mon): 日本古代的长度单位，也曾是货币单位。在制鞋业中，1文约等于2.4厘米。这个单位至今仍在一些传统鞋店（如木屐店、足袋店）中使用，是传统工艺传承的体现。

## 各种各样的足袋

好了好了，除了上述那些，还有以下这些足袋。全部都是男士款式。顺带一提，这些我全都有，但基本上没怎么穿过就是了。(^^;

*   **别珍足袋**
    昭和初期曾一度流行，主要用作防寒。其鞋底也使用了别珍面料。这本是女性足袋的主流，过去在超市的内衣区也能见到，但如今除了专业的足袋店，已基本销声匿迹。颜色选择相当丰富。

*   **麻足袋**
    夏季用的时尚足袋。由100%的麻制成，夏天穿着轻便凉爽，但缺点是极易起皱，同时脚趾的轮廓会清晰地显露出来，因此很难穿得好看。选择它时，比选棉质足袋更需要尺寸的绝对贴合。福助等品牌也有生产。

*   **弹力足袋**
    由尼龙材质制成的、可伸缩的足袋。其面料非常柔软，以至于金属扣襻硌在脚上都会觉得疼，因此脚趾的形状也同样容易显现。不过优点是洗后直接晾干即可，打理起来很方便。

*   **云斋足袋**
    鞋面和鞋底均由厚实耐用的棉布制成，是匠人们在冬季等时节常穿的款式。照片中的是福助制造的，其鞋底采用了被称为“石底”的加厚棉布。
    *   **[补充知识]** 一般来说，足袋底主要有三种：布面呈斜向条纹的“**云斋底**”[^23]:、呈W字形斜纹织法的“**杉底**”[^24]:，以及表面凹凸不平的“**石底**”[^25]:。此外，似乎有这样一种普遍的地域习惯：关西地区多用“杉底”，而关东地区则惯用“云斋底”。

[^23]:云斋底 (雲斎底 / Unsaizoko): 采用称为“云斋织”的厚实斜纹棉布制成，特点是耐磨、防滑，布面有清晰的斜向纹理。在关东地区较为流行。
[^24]:杉底 (杉底 / Sugizoko): 采用称为“缀织”（Tsuzure-ori）的织法，布面呈现出类似杉树叶排列的W形或人字形纹理，因此得名。质地比云斋底略软，在关西地区较为常见。
[^25]:石底 (石底 / Ishizoko): 指使用极厚、表面粗糙如石的棉布制成的鞋底，耐磨性最强，主要用于需要高强度劳作的场合，如匠人或祭典抬轿人员所穿的足袋。

## 关于足袋的保养

清洗足袋时，将其放入洗衣网，直接用洗衣机洗涤完全没有问题。但需注意，全新的彩色足袋可能会掉色，尤其是绀色足袋，似乎更容易掉色。

<Image src="https://ik.imagekit.io/dgfwhfvkn/tabi/washtabi_waifu2x_noise3_scale4x_xbur_qYPQ.avif?updatedAt=1739357473795" client:visible />

晾晒时，要充分拍打足袋的脚背和鞋底部分，将褶皱抚平后再晾。用铁丝衣架改造一下，将足袋直接套在上面晾晒，效果绝佳。如果使用洗衣夹，务必夹在挂扣襻用的“挂线”[^26]那一侧的边缘。这样即便留下夹痕，也不会显眼。顺带一提，足袋有时会变形，所以最好不要翻过来晾。

[^26]:挂线 (かけ糸 / Kakeito): 指足袋脚踝内侧，与金属扣襻（小鉤）相对应的一排或多排线圈。通过将扣襻挂入不同的线圈，可以微调足袋的松紧度。

晾干后，无需熨烫。直接穿上，用自己脚的形状将褶皱自然撑平即可。因为熨烫会导致面料起光。

此外，全新的足袋表面有浆，初次穿着前，最好先下水洗一次，将浆洗掉，这样会更贴合脚型。如果觉得下水洗麻烦，也可以直接穿着新足袋去洗个澡。

要知道，比起崭新的足袋，反复洗涤、穿旧了的足袋，其穿着感会好得多。正因如此，我常常对那些被我穿得破破烂烂、却已与我的脚型完美契合的足袋，百般爱惜，不舍得丢弃，最后都留作了居家鞋。

顺便说一下，我的情况是，小脚趾根部的外侧最容易磨破。如果足袋尺寸过紧，脚尖或脚掌侧面就很容易磨损。归根结底，这取决于穿着足袋时的具体状况，但总归是那些容易受到摩擦、或总是受力的部位更容易破损。

当然，只要还能修补，我都会补了再穿。但说到底，足袋终究和内衣一样，或许是应该被看作消耗品的物件吧。

# 足袋的穿法

足袋，也是有其正规、正确的穿法的。请通过下面来确认一下吧。

据说在过去，当大多数学生还穿着和服上学的时代，不扣足袋的扣襻[^27]（这感觉大概就像现在踩着运动鞋的后跟走路？），或是任由袴的后系带[^28]垂着，都会被视为衣冠不整而受到提醒。

[^27]:扣襻 (こはぜ / Kohaze): 指足袋脚踝内侧用于固定的金属或骨制挂扣。不扣扣襻，就如同穿鞋不系鞋带，是典型的衣着不整的表现。

[^28]:袴的后系带 (袴の後紐 / Hakama no ushiro-himo): 指穿着袴时，在背后交叉后于身前打结的系带。任其垂着而不系好，同样是严重违反着装规范的行为。

我想现在应该不会有这样的人了，但还是要充分注意：一双没有好好穿在脚上的足袋，不仅会加速足袋本身的损耗，还有可能伤到自己的脚。

## 穿着足袋的姿势

从穿着顺序上来说，本应在穿好兜裆布（褌）[^29]或男士衬裤（ステテコ）[^30]等遮蔽下半身的贴身衣物之后，立刻就穿上足袋。即便是需要穿着贴身汗衫（肌襦袢）[^31]或下摆衬裙（裾よけ）[^32]的场合，也应在穿好足袋之后再进行。这是因为，如果将穿足袋的步骤留到最后，很可能会导致已经穿好的衣物形态散乱。因此，至少，请在穿着长衬衫（長襦袢）[^33]之前，就将足袋穿好。

<Image src="https://ik.imagekit.io/dgfwhfvkn/tabi/w_tabi_1_waifu2x_noise3_scale4x_HQIl3diK4.avif?updatedAt=1739365098318" client:visible />

[^29]:兜裆布 (褌 / Fundoshi): 日本男性的传统贴身内裤，通常为一条长棉布，有多种缠绕和打结方式（如六尺褌、越中褌等）。
[^30]:男士衬裤 (ステテコ / Suteteko): 一种宽松的、长度及膝的男士衬裤，主要用于夏季，穿在和服（長着）之内，以防汗水浸湿外衣并保持凉爽。
[^31]:贴身汗衫 (肌襦袢 / Hadajuban): 穿在最里层、直接接触皮肤的贴身汗衫，主要功能是吸汗，防止汗水弄脏上层的长襦袢和着物。
[^32]:下摆衬裙 (裾よけ / Susoyoke): 一种围在腰间的片状衬裙，用于保护长襦袢和着物的下摆部分不被弄脏，并有助于行走时衣物的顺滑。
[^33]:长衬衫（長襦袢 / Nagajuban）: 长衬衫 (長襦袢 / Nagajuban): 穿在肌襦袢之外、着物之内的一件长款衬衣。它的领子（半衿）会从着物的领口露出，是构成整体造型的重要部分，同时也起到了隔离内外衣物的作用。

## 足袋的穿法

1.  如左图所示，先将足袋的上半部分向外翻折约一半。
2.  将翻折好的足袋套在脚上，然后用双手向自己的方向轻轻拉拽，确保脚趾能完全伸入并抵达足袋的最前端。
3.  接着，将翻折的部分复原，使其完全包裹住脚踝，直至脚后跟。

## 扣襻的扣法

1.  **顺序**: 扣扣襻时，**必须从下往上**，依次将每一个扣襻挂入对应的线圈（かけ糸）中。

2.  **线圈的选择**: 每个扣襻位通常都配有两根线圈。扣的时候请注意，朝向身体内侧的那根是**备用线圈**。将扣襻挂在外侧的线圈上，才是正确的扣法，此时的松紧度，也才是这双足袋的**标准尺码**。因此，在挑选足袋尺寸时，务必以扣上外侧线圈为准。

3.  **技巧**: 如果感觉扣襻很难扣上，可以尝试抓住脚踝，向外或向内轻轻转动，利用角度的变化来辅助扣紧。

4.  **脱法**: 顺带一提，脱足袋时则无需讲究顺序，随意脱下即可。
