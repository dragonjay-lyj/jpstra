## 前言

“褌”（Fundoshi），即兜裆布，是日本的传统内衣。然而在当今的日本，恐怕很多人只在历史剧或祭典上才见过它。正如和服已不再是日常衣物一样，自明治维新推行西化着装政策以来，兜裆布也已彻底退出了日常生活的舞台。

但是，在今天，兜裆布的价值正被重新审视，其爱好者也在不断增加。

究其原因，在于兜裆布的功能性与美学意识得到了新的认可。换言之，它那与西式内裤² 截然不同的构造、功能及设计，正被在西式内裤环境中成长起来的年轻一代，以一种新奇的眼光所接纳。

与西式内裤相比，兜裆布的特征大致可归纳为以下几点：

一、由非弹性材质制成。
二、以长方形布料为基本形态。
三、需根据自身体型进行调节后穿戴。
四、在某些场合，可作为正式服装（而非内衣）穿着。

那么，相较于西式内裤，兜裆布的优点又有哪些呢？这自然与上述四个特征密切相关。

甲、可根据个人体型、当日身体状况及心情，自由调节松紧度。
乙、通过调节松紧，可控制其透气性与贴合度，以适应气候及温湿度的变化。
丙、因使用非弹性布料，能获得一种独特的开放感，抑或是一种紧绷感。³
丁、通过“系结”⁴ 的方式穿戴，在某些情况下，比起“套穿”的西式内裤，穿脱更为便捷。
戊、可以享受布料的材质、颜色、花纹等丰富多样的变化所带来的乐趣。

以上优点，再次与和服及西式服装的区别呈现出相似性。由此可见，在日常生活中使用兜裆布所面临的问题，也必然与穿着和服时遇到的问题相似。

在日常生活中使用兜裆布的问题点：

2.  难以买到。即便能买到，价格也可能比较昂贵。
3.  不知道正确的穿戴方法，周围也无人知晓。
4.  由于现代生活以西式服装为标准，有时会带来不便。

然而，这些都算不上是真正的问题。

### ・六尺褌 (Roku-shaku Fundoshi)

穿上后感觉紧致挺括，富有张力。祭典上常见的兜裆布造型，多为六尺褌。其系法需要一些技巧。

### 关于六尺褌

六尺褌是构造最为简单的兜裆布，本质上就是一条长布。

其名“六尺”，源于日本传统长度单位“鲸尺”¹，一根长约六尺（37.88厘米 × 6 ≈ 230厘米）的布料，缠绕于腰间。

六尺褌的系法主要有两种：一种是留有前垂布的系法（下称“有前垂系法”），另一种是将前垂部分穿过裆部、固定于身后的系法（下称“无前垂系法”）。

“有前垂系法”可以轻松地将兜裆布放松，方便如厕。
“无前垂系法”则能系得非常牢固紧实。在身体活动剧烈的祭典等场合，这种系法更为普遍。

六尺褌通常在普通商店里没有售卖。一般是购买“晒木棉”² 布料，根据自己的身材裁剪使用。关于晒木棉的购买方法，请参阅“购买兜裆布”一节。
布料的长度需根据体型调整，大致标准如下：若采用“有前垂系法”，长度约为“腰围×3”再略短20厘米左右；若采用“无前垂系法”，则需比“腰围×3”稍长一些。

它与“越中褌”最大的区别在于，**六尺褌穿好后，臀部会呈现出T字裤（丁字裤）的状态**，并且比越中褌能系得更紧、更牢固。
如果说越中褌是一种“宽松”的兜裆布，那么六尺褌便可称之为“**紧绷**”的兜裆布。当然，其松紧度完全可以自由调节。**而臀部的完全裸露，更能带来其他任何内衣都无法给予的、独特的穿着感**。³

直到明治时期，六尺褌都是日本男性的标准内衣。在葛饰北斋等人的浮世绘或“洒落本”⁴ 插图中，市井之人的内衣大多是六尺褌。
然而，从江户时代末期开始，越中褌在医生、知识分子及老年人群体中开始普及，六尺褌的地位逐渐被越中褌所取代。
总的来说，在江户时代，可以大致认为：

*   **越中褌 = 宽松 = 非劳动时穿着的内衣**
*   **六尺褌 = 紧绷 = 劳动时穿着的内衣**

当然，作为现代人，我们可以完全根据个人喜好来选择。我自己就会根据当天的心情，交替使用越中褌和六尺褌。

### 六尺褌的系法

此方法的部分步骤与“有前垂系法”相同。

1.  预留比“有前垂系法”更长的布料搭在肩上，另一端从裆下穿过。

<Image src="https://ik.imagekit.io/dgfwhfvkn/tabi/fastening_ryokusyaku5_waifu2x_noise3_scale4x_dTzRPDpHM.avif?updatedAt=1739713708132" client:visible/>
    *(图示1：将更长的一段布料搭在肩上作为预留，另一端从裆下穿过)*

2.  将绕到身后的布料主体在腰间缠绕，然后将布料末端从臀部中心（股缝）处由下往上穿出，并塞入腰间的横向布带内。向上拉紧这根纵向布带，以整理形态并调节松紧度。
    *(此步骤与“有前垂系法”的步骤2相同)*

<Image src="https://ik.imagekit.io/dgfwhfvkn/tabi/fastening_ryokusyaku2_waifu2x_noise3_scale4x_J8liaLQou.avif?updatedAt=1739713705509" client:visible/>
    *(图示2：同上)*

3.  将穿出的布料末端翻转下垂，然后绕着腰部的横向布带拧转缠绕一至两圈。再将剩余的布头塞入横带中固定好。
    *(此步骤与“有前垂系法”的步骤3相同)*

<Image src="https://ik.imagekit.io/dgfwhfvkn/tabi/fastening_ryokusyaku3_waifu2x_noise3_scale4x_wpgzMJgiWt.avif?updatedAt=1739713708960" client:visible/>
    *(图示3：同上)*

4.  将身前预留的布料（原搭在肩上的部分）穿过裆下，然后绕着身后那根纵向的布带（即原文的“たてまわし”）² 缠绕数次，一直向上。之后，按照步骤3同样的方法处理布料末端。

<Image src="https://ik.imagekit.io/dgfwhfvkn/tabi/fastening_ryokusyaku8_waifu2x_noise3_scale4x_EFLplhTp7l.avif?updatedAt=1739713705883" client:visible/>
    *(图示4：将前方的长布穿过裆下，并开始缠绕在后方的纵向布带上)*

5.  整理好前方的“囊袋”³，使其宽度比“有前垂系法”略窄一些即可。

<Image src="https://ik.imagekit.io/dgfwhfvkn/tabi/fastening_ryokusyaku7_waifu2x_noise3_scale4x_oqmQNNrpf.avif?updatedAt=1739713705543" client:visible/>
    *(图示5：完成后的正面造型，没有前垂布，只有一个包裹住重点部位的囊袋)*

**（注）**
日本“尺贯法”中的长度单位，主要分为用于建筑领域的“**曲尺 (Kane-jaku)**”和用于裁缝领域的“**鲸尺 (Kujira-jaku)**”。

*   **曲尺**：一尺 ≈ 30.3厘米
*   **鲸尺**：一尺 ≈ 37.88厘米

两者差异相当大。虽然曲尺是更通用的单位，但由于兜裆布属于衣物，故使用鲸尺。
“尺”存在两种标准这件事，出人意料地鲜为人知，以至于在一些正式出版物中，也能见到误用的情况。
